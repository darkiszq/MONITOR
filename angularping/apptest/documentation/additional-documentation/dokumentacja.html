MONITOR - Dokumentacja

WPROWADZENIE

Aplikacja MONITOR to strona internetowa napisana w frameworku Angular połączona z bazą danych i serwerem Express.js, służąca do monitorowania aktywności serwerów stron internetowych oraz weryfikacji ich odpowiedzi na wysyłane pakiety.

FUNKCJE

  • Wyświetlanie aktualnego statusu określonych stron (OFFLINE bądź ONLINE)
  • Wyświetlanie wykresu kołowego pokazującego aktywność stron na przestrzeni całego czasu działania aplikacji
  • Zapisanie raportu z aktywności obserwowanych stron na urządzeniu użytkownika
  • Zapis aktualnego statusu strony w bazie danych

ZASTOSOWANE TECHNOLOGIE

Backend

  • Node.js
    • Express.js
    • Mysql2

Frontend

  • Angular
    • Rxjs
    • Express.js
    • Charts.css

Baza danych

  • XAMPP

INSTALACJA

WYMAGANIA

  • System operacyjny Windows 11
  • Angular CLI (wersja 21.0.4)
  • Node.js (wersja 24.12.0)
  • GitHub CLI (wersja 2.83.2), bądź jakikolwiek program umożliwiający klonowanie repozytorium GitHub
  • XAMPP (wersja 3.0.0)
  • Działające połączenie z internetem + przeglądarka internetowa

PROCES INSTALACJI

WSTĘP

  1. Stwórz nowy pusty folder w wybranym przez siebie miejscu

  2. Sklonuj repozytorium do folderu

    Jeśli używasz GitHub CLI:

    • Otwórz terminal w folderze przeznaczonym na aplikację
    • Jeśli używasz GitHub CLI po raz pierwszy wpisz komendę:Example :
      gh auth login
      i zaloguj się postępując zgodnie z instrukcjami w terminalu
    • Wpisz komendę:Example :
      gh repo clone darkiszq/MONITOR

  3. Upewnij się że lokalne porty 4200, 8080 i 3306 nie są blokowane, to na nich działa aplikacja

BAZA DANYCH

  1. Otwórz program XAMPP
  2. Włącz serwer Apache i MySQL
  3. Kliknij kontrolkę "Admin" przy MySQL
  4. Na lewym pasku kliknij przycisk "New"
  5. Wpisz nazwę "uptime" i kliknij przycisk "Create"
  6. Przejdź do zakładki "Import"
  7. Wybierz plik "uptime.sql" ze sklonowanego wcześniej folderu i kliknij przycisk "Import" nie zmieniając ustawień importu

URUCHOMIENIE APLIKACJI

  1. Powtórz dwa pierwsze kroki z poprzedniego punktu
  2. Otwórz dwa Wiersze Poleceń (CMD)
  3. Wpisz w jednym CMD komendę
Example :
cd [LOKALIZACJA FOLDERU MONITOR]/angularping/apptest
  1. Wpisz komendę
Example :
npm install
  1. Wpisz komendę
Example :
ng serve --proxy-config proxy.conf.json
  1. Otwórz drugi CMD i wpisz w nim komendę
Example :
cd [LOKALIZACJA FOLDERU MONITOR]/databasepostserver
  1. Wpisz komendę
Example :
node index.js
  1. W przeglądarce internetowej wejdź na adres http://localhost:4200/
